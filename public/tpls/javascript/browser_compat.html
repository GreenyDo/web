		能力检测
		根据浏览器不同将能力组合起来是更可取的方式。
		怪癖检测：想要知道浏览器存在什么缺陷。
		用户代理检测：通过检测用户代理字符来确定实际使用的浏览器，在每一次HTTP请求过程中，用户代理字符串是作为响应首部发送的而且该字符串可以通过javascript的navigator.userAgent属性访问。
		浏览器检测顺序：
		1.Opera    // Opera
		2.WebKit  // Chrome、Safari、
		3.KHTML  //Konqueror(Linux)
		4.Gecko   //Firefox
		5.IE           //
		//完整检测代码《javascript高级程序设计》第三版p242
		优先使用能力检测和怪癖检测，用户代理检测是客户端检测的最后一个选择

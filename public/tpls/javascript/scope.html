apply()和call()


		用途都是在特定的作用域中调用函数，实际上等于设置函数体内this对象的值
		apply();接收两个参数
		1.在其中运行函数的作用域
		2.参数数组；//可以是Array的实例，也可以是arguments对象
		eg.
		function sum (num1,num2) {
		        return num1+num2;
		}
		function callSum1(num1,num2) {
		        return sum.apply(this, arguments);//this 是window
		}
		function callSum2(num1,num2) {
		        return sum.apply(this, [num1,num2]);
		}
		call()与apply()函数的不同之处在于，接收参数的方式不同。
		传递给函数的参数必须逐个列举出来
		真正强大的地方是能够扩充函数赖以运行的作用域
